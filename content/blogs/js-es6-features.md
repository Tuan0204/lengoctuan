## Ví dụ nhanh


ES6 mang đến nhiều cú pháp hiện đại giúp mã ngắn gọn và dễ bảo trì.

## Ví dụ nhanh

```js
const add = (a, b) => a + b;
const [x, y] = [1, 2];
console.log(`${x} + ${y} = ${add(x,y)}`);
```

## Lời khuyên

- Dùng const cho hằng số, let cho biến thay vì var
- Sử dụng module để tách rời mã
